# Hicord

Hicord は、「位置」の概念を取り入れた新感覚のビデオチャットツールです。

使いやすさを追及した様々な新機能を備えています。

## Getting Started

### 初期インストール

#### Node.js のインストール

#### pnpm のインストール

```bash
npm install -g pnpm
pnpm add -g pnpm
```

[Volta](https://volta.sh/) を使用する場合

```bash
volta install pnpm
pnpm add -g pnpm
```

#### Dependencies のインストール

```bash
pnpm install
```

#### .env の設定

1. `.env.example`ファイルを複製
2. ファイル名を`.env.local`に変更
3. API キーを入力する

### 実行

```bash
pnpm dev
```
